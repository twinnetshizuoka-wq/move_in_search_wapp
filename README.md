# 賃貸・購入物件 入居発見ツール

プログラミングの知識がなくても使えます。  
**コマンド入力は不要**です。Cursor や VSCode も不要です。

古い物件データと新しい物件データを比べて、掲載が消えた物件を「入居済み候補」として見つけます。

---

## ダウンロード（これだけでOK）

配布ファイルは **zip 1つだけ** です。

1. [GitHub の Releases](https://github.com/twinnetshizuoka-wq/move_in_search_wapp/releases/latest) から  
   **`入居発見ツール.zip`** をダウンロード
2. zip を右クリック → **すべて展開**
3. 出てきたフォルダを開く

※ 開発者向けにリポジトリ全体を見る場合は [GitHub](https://github.com/twinnetshizuoka-wq/move_in_search_wapp) を開いてください。

---

## 使い方（ダブルクリックだけ）

### 1. アプリを起動する

フォルダ内の **`スタート.bat`** をダブルクリックします。

- **初回だけ**、必要な部品を自動で用意します（数分かかります）
- 黒い画面に文字が出ても、そのまま待ってください
- 準備が終わると、アプリの画面が開きます

### 2. 物件情報を取得する

1. アプリで **賃貸情報を取得** または **購入情報を取得** をクリック
2. 開いたブラウザで、調べたい市区町村の物件一覧を表示する
3. 黒い画面（ターミナル）をクリックして **Enter キー** を押す
4. 取得が終わると、`data` フォルダに CSV が保存されます

同じ条件で、あとでもう一度取得しておくと、比較ができます。

### 3. 入居済み候補を比較する

1. **`比較.bat`** をダブルクリック（またはアプリの「比較」）
2. **古いファイル** と **新しいファイル** を選ぶ
3. **比較を実行** をクリック
4. 結果は `data` フォルダに保存されます

### 4. 他社比較する（アットホーム × HOME'S）

1. アプリの **他社比較** をクリック
2. アットホームのCSVと HOME'S のCSVを選ぶ
3. **他社比較を実行** をクリック
4. 同じ物件名の物件情報が `data` フォルダに保存されます（住所は比較しません）

## はじめての注意（Python）

このツールは内部で Python を使います。

- PC に Python が入っていない場合、`スタート.bat` が自動で案内します
- 表示されたページから **Python 3.10 以上** をインストールしてください
- インストール画面で **「Add python.exe to PATH」に必ずチェック** を入れてください
- インストール後、もう一度 `スタート.bat` をダブルクリックしてください

初回起動時に、次も自動で行われます。

- `requirements.txt` のインストール
- Playwright（ブラウザ自動操作用）のインストール

手作業でのコマンド入力は不要です。

## 画面のボタン説明

| ボタン | 内容 |
|---|---|
| 賃貸情報を取得 | アットホーム / HOME'S の賃貸一覧を取得 |
| 購入情報を取得 | アットホーム / HOME'S / しずナビの購入一覧を取得 |

## よくある質問

### 黒い画面が出て止まりました

初回セットアップ中です。数分待ってください。  
エラーで止まった場合は、表示されたメッセージを読んでください。

### Enter を押しても始まらない

物件の **一覧ページ** を表示してから Enter を押してください。  
サイトのトップページのままでは開始できません。

### CAPTCHA（ロボット確認）が出た

ブラウザ上で確認を完了してから、もう一度 Enter を押してください。

### 比較でエラーになる

- 同じエリア（都道府県・市区町村）のファイル同士を選んでください
- 古いファイルの日付が、新しいファイルより前であることを確認してください

### Cursor や VSCode は必要ですか？

不要です。zip を展開して `スタート.bat` をダブルクリックするだけで使えます。

## 開発者向け（一般の方は不要）

### ローカルで開発する場合

```powershell
pip install -r requirements.txt
playwright install chromium
python app.py
```

### 配布用 zip を作る場合

`配布用zipを作成.bat` をダブルクリックします。

作成先:

- `dist/入居発見ツール.zip`（配布用の本体）
- `dist/nyuukyo-hakken-tool_YYYYMMDD.zip`（日付付きコピー）

この zip を GitHub Releases にアップロードしてください。ユーザーがダウンロードするファイルはこれ1つです。
